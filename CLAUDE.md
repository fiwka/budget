# Budget - информационная система для анализа и планирования личного бюджета

## Что это?

Данный проект представляет собой курсовую работу по теме "Разработка информационной системы для анализа и планирования личного бюджета".

Помимо обычного статистического анализа бюджета, здесь предполагается внедрение ИИ-агента

## Стек
Java 25, Spring Boot 4, PostgreSQL, Apache Kafka, Docker, GitHub Actions

## Архитектура

DDD + clean architecture (usecase, port, adapter (infrastructure))

## Описание подпроектов

`ai-agent` - API помощника соединение MCP сервера и LLM, стриминг на клиент

`mcp-server` - MCP сервер для работы с системой

`data-service` - CRUD для основных данных: бюджеты, категории транзакций, транзакции, связи между пользователями и единицами бюджета и самим бюджетом

`analytics-service` - аналитический сервис принимает сообщения от `data-service` о создании транзакции и просчитывает необходимые аналитические метрики

`api-gateway` - API Gateway который светит наружу, авторизация и обращение к `eureka-server` для Service discovery

`eureka-server` - сервер Netflix Eureka для Service Discovery

`migrations` - миграции баз данных для аналитики и данных
